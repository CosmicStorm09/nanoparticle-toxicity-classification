# Nanoparticle Toxicity Classification using Machine Learning

## Overview
This project builds a machine learning model to classify nanoparticles as toxic or non-toxic based on their physicochemical properties.

## Methodology
- Data preprocessing (handling missing values, encoding, scaling)
- Model: Random Forest Classifier
- Hyperparameter tuning using GridSearchCV
- Evaluation using accuracy, confusion matrix, and classification metrics

## Results
- Accuracy: ~95%
- Key factors influencing toxicity:
  - Dosage (most important)
  - Oxygen-related properties
  - Exposure time and surface area

## Visualizations
- Confusion Matrix
- Feature Importance Plot

## Key Insight
Higher dosage and oxidative interactions significantly increase nanoparticle toxicity.
